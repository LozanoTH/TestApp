# TestApp

![logo](assets/images/icon.png)

**TestApp** es una aplicación para probar el funcionamiento y las capacidades de **React Native** con **Expo** en dispositivos Android.

## Características
- Configuración rápida con Expo.
- Pruebas de componentes nativos de React Native.
- Compatible con Android (y opcionalmente iOS).
